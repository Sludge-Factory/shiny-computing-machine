Basics of HTML

This is the section where you will learn the basics of this language. The language of HTML is broken down into small sections such as Headers (h1, h2, h3, etc), Paragraphs (p), and Dividers (div). These units each serve a particular function and the compiler within browsers that the code runs through automatically use these units to format your code into webpages.
